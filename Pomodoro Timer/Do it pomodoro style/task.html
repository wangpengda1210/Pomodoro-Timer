Description

In this stage, you should implement your custom view (name this class TimerView and put it into org.hyperskill.pomodoro.timer package).

This class should contain String getText() and int getColor() methods.

It should work as follows:

  1. TimerView should do N "work" rounds and N-1 "rest" rounds and then stop.
  2. "work" and "rest" rounds should follow one another, starting from "work" round.
  3. TimerView is red during a "work" round
  4. TimerView is green during a "rest" round
  5. After the last "work" round TimerView is yellow

You may implement an animation of this TimerView, like a circle in the image below, but the easiest way is just filling up the space of TimerView with plain color.

Replace TextView from last steps to TimerView, with id timerView.
